Introduction

This is an Practical Machine Learning course projest. The assignment objective is to build the learning algorythm to the Human Activity Recognitiondata set of 6 participants who were asked to perform barbell lifting activities correctly and incorrectly in 5 different ways.

Data set credits: Velloso, E.; Bulling, A.; Gellersen, H.; Ugulino, W.; Fuks, H. Qualitative Activity Recognition of Weight Lifting Exercises. Proceedings of 4th International Conference in Cooperation with SIGCHI (Augmented Human 13) . Stuttgart, Germany: ACM SIGCHI, 2013.

Read more: http://groupware.les.inf.puc-rio.br/har#ixzz3JtzTNxAi

Processing Data

Reading the raw data from CSV files and building the training model on the classe factor against the non-empty columns of the training set: roll_belt, pitch_belt, yaw_belt, gyros_belt_x, gyros_belt_y, gyros_belt_z, accel_belt_x, accel_belt_y, accel_belt_z, magnet_belt_x, magnet_belt_y, magnet_belt_z, roll_arm, pitch_arm, yaw_arm, total_accel_arm, gyros_arm_x, gyros_arm_y, gyros_arm_z, accel_arm_x, accel_arm_y, accel_arm_z, magnet_arm_x, magnet_arm_y, magnet_arm_z, roll_dumbbell, and yaw_dumbbell.

Were using the Random Forest algorithm for the training model.

## Warning: package 'caret' was built under R version 3.1.1
## Loading required package: lattice

rawTrain = read.csv ("pml-training.csv")
rawTest = read.csv ("pml-testing.csv")

fit = train (classe ~ roll_belt + pitch_belt + yaw_belt + gyros_belt_x + 
                     gyros_belt_y + gyros_belt_z + accel_belt_x + accel_belt_y + 
                     accel_belt_z + magnet_belt_x + magnet_belt_y + magnet_belt_z + 
                     roll_arm + pitch_arm + yaw_arm + total_accel_arm + gyros_arm_x + 
                     gyros_arm_y + gyros_arm_z + accel_arm_x + accel_arm_y + accel_arm_z + 
                     magnet_arm_x + magnet_arm_y + magnet_arm_z + roll_dumbbell + 
                     yaw_dumbbell, method = "rf", data=rawTrain)
